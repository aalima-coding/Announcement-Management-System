


#  Announcements Management Project 
A Full-Stack MERN CRUD Application

This project is a simple yet complete demonstration of a full-stack CRUD (Create, Read, Update, Delete) system for managing announcements.  
Users can create announcements with a title and description, view them instantly, edit, or delete them â€” all from the frontend, with data stored securely in MongoDB Atlas.

---

## Features

 -Create new announcements with title and rich-text description  
 -View all announcements in a clean responsive UI  
 -Update existing announcements  
 -Delete announcements instantly  
 -RESTful API structure (GET, POST, PUT, DELETE)  
 -Real-time frontend integration using React + Fetch API  
 -MongoDB Atlas connection for cloud database  
 -Clean modular code with Mongoose schema and Express routing

---

##  Tech Stack

| Layer | Technology |

| **Frontend** | React (Vite) + Fetch API |
| **Backend** | Node.js + Express.js |
| **Database** | MongoDB Atlas (via Mongoose) |
| **Language** | JavaScript (ES6 Modules) |
| **Tools** | Postman, VS Code, npm |
